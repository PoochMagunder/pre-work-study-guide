HTML Notes

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • (H1-H6): sets the header to different sizes.
  • (P): identifies a paragraph or block of text.
  • (UL): identifies an "unordered list".
  • (OL): identifies an "ordered list".
  • (LI): identifies a listed item inside a "UL" or "OL".
  • (IMG): identifies an "image".
  • (SRC): identifies a "web link or image source".
  • (BR): identifies an "empty space or break"
  • (A): anchors a link.
  • (HREF): points to a URL.

Git Notes

  • Git status: checks what branch we are currently on.
  • Git checkout -b "branch -name": Creates a new branch and switches to it.
  • Git add -A: Commits all files in the working branch.
  • Git commit -m: Sends the updated file to the repo with a message.
